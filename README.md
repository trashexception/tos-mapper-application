## 처음 실행하세요?
1. [옵션] → [맵 이름 영역 지정]
![옵션 → 맵 영역 지정](/image/option.png)
1. 영역 __시작__ 클릭 → 영역 __끝__ 클릭 → [저장] → [닫기]
![영역 시작 클릭 → 영역 끝 클릭 → 저장 → 닫기](/image/area.png)
1. 자동 갱신 클릭
![자동 갱신 클릭](/image/reflash.png)
1. PLAY & ENJOY
![PLAY & ENJOY](/image/play.png)
